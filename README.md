# bis
Not one game engine, but the same, again.

- [ ] `orderstone` A Minecraft clone? The idea is that multiple distinct "chunksets" can exist, which can act as physics objects independent of world terrain. So you could build an airship, and continue to build on it as it is moving.
- [ ] `arcflash` Kodu Game Lab/Project Spark clone. Enough said.
- [ ] `biscuit` A more general-purpose engine.

- `tailmaw` Shared code and resources across all engines.
